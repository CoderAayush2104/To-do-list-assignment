
# To Do List

Technologies used : HTML,CSS,Javascript,React,and Redux.


## Run Locally

Clone the project

```bash
  git clone https://github.com/CoderAayush2104/To-do-list-assignment
```

Go to the project directory

```bash
  cd To-do-list-assignment
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
